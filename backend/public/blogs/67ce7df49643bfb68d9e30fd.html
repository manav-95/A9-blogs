Amazon Online Assessment Front-End Engineer — JavaScript, HTML, CSS

This question was asked in the month of September 2024. I received Hackerrank link for the Amazon’s online assessment. This time around the Front-End assessment are getting conducted in more front-endy fashion.

There will be Two questions that will be asked in 2 hours time slot. You will be expected to solve the questions using JavaScript, HTML, CSS.

Question 1:

Implement a simple accordion (expand and collapse) behavior for the given layout so user can lean about their book collection.

Technology: HTML, CSS and JS

Requirements:

  • All item titles are visible at all times.

  • The first item is in the expanded state by default.

  • When an item is in the expanded state, the collapse icon (-) is visible next to the title.

  • When an item is in the collapsed state, the expand icon (+) is visible next to the title.

  • Clicking on the title of the expanded item collapses it and expands the selected item.

  • Clicking on the title of a collapsed item expands it.

  • If the “Multiple” checkbox is checked, it’s possible to expand more than one item.
